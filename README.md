# 研发效能深度分析看板 (R&D Efficiency Analytics Dashboard)

这是一个基于 React + TypeScript + Vite + ECharts 构建的纯前端、高性能的研发效能数据分析看板。它可以将散乱的 CSV 代码统计数据，转化为直观的多维分析图表，并集成 LLM 智能洞察，帮助团队快速掌握产出动态。

## 🌟 核心特性

- **🚀 零配置一键启动**：自带 Windows 批处理启动脚本 (`start_dashboard.bat`)，一键安装依赖并启动服务。
- **🤖 AI 智能分析**：集成 LLM（如 DeepSeek, OpenAI），支持按全工程、特定工程或特定人员生成深度效能报告，支持自定义分析焦点。
- **🌐 局域网无缝共享**：服务启动后自动绑定 `0.0.0.0`，同局域网同事可直接通过 IP 地址免安装查看最新数据。
- **📂 自动目录感知**：默认自动扫描并加载根目录下各文件夹中的 `.csv` 文件，自动按文件夹归类为“顶级工程”。
- **👥 人员身份归一化**：通过 `author_mapping.json` 自动合并不同工号、别名到统一真实姓名。
- **📊 全维度数据可视化**：
  - **工程维度**：多工程合集统计、子模块活跃度分析。
  - **月度对比**：人员横向月度产出堆叠图，支持人员与月份的双重过滤。
  - **个人明细**：专属产出趋势折线图与最细粒度的月度流水账。
  - **指标切换**：实时切换“新增行数”、“删除行数”、“净增行数”视图。

## 🛠️ 快速开始

### 前置条件
确保您的电脑上已安装 [Node.js](https://nodejs.org/) (推荐 v18+)。

### Windows 推荐启动方式
双击项目根目录下的 **`start_dashboard.bat`**。
脚本将自动处理环境检查、依赖安装（首次）、服务启动及局域网 IP 提示。

### 通用命令行启动
```bash
npm install
npm run dev
```

## 🤖 AI 智能分析使用指南

在看板的 **“AI 分析”** 标签页中，您可以获得专家级的效能洞察：

1. **配置**：填入您的 API Key（默认支持 DeepSeek 接口，也可自定义 Endpoint）。
2. **模式选择**：
   - **全工程概览**：宏观评估整体研发态势。
   - **特定工程深度分析**：聚焦核心项目的资源投入合理性。
   - **特定人员效能评估**：针对性地评价核心开发者的贡献稳定性。
3. **自定义焦点**：在输入框中填入您关心的问题（如：“分析为什么 2 月份产出有所下滑？”），AI 将重点解答。

## 📁 数据准备与身份归一化

### 目录约定
```text
根目录/
├── author_mapping.json   <-- 人员映射配置
├── 项目A_文件夹/          <-- 自动识别为工程名
│   └── stats.csv
├── 项目B_文件夹/
│   └── backend_stats.csv
└── start_dashboard.bat
```

### 身份归一化 (author_mapping.json)
```json
{
  "工号1": "真实姓名",
  "拼音名": "真实姓名"
}
```
配置后刷新页面即可看到合并后的统计结果。
